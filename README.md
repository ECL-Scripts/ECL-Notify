# ECL-Notify

Modern and minimal notification system for FiveM built with smooth animations, optimized performance and premium UI design.

━━━━━━━━━━━━━━━━━━━━━━

## Features

• Modern dark UI
• Smooth animations
• Notification sound effects
• Success / Error / Warning / Info notifications
• Optimized performance
• Lightweight design
• Standalone support
• Easy integration
• Clean ecosystem styling

━━━━━━━━━━━━━━━━━━━━━━

## Compatibility

• Standalone
• ESX
• QBCore
• ox_core

## Installation

Add the resource to your `resources` folder and ensure it in your `server.cfg`.

```cfg id="jlwmec"
ensure ECL-Notify
```
